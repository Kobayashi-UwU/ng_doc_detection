## Project Detail
This is a document detection outline using OpenCV and the Yolo model to detect and classify types of documents using these classes.
```
'bank_acc',
'card_id',
'copy_bank_acc',
'copy_card_id',
'copy_driving_license',
'copy_passport',
'driving_license',
'glasses',
'hat',
'mask',
'passport',
'sunglasses',
```

## How to start project
```
npm install
ng serve
```
